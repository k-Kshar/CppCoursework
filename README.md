# Employee Management System (C++)

## Особенности
- Используется ООП (наследование, полиморфизм)
- Работа с динамической памятью
- Обработка ошибок через исключения
- 
## Описание
Консольное приложение для управления сотрудниками компании.

Реализованы:
- добавление сотрудников
- удаление сотрудников
- редактирование данных
- расчет зарплаты
- поддержка разных ролей (директор, бухгалтер, программист и т.д.)

## Технологии
- C++
- ООП (наследование, полиморфизм)
- STL (vector, map)

## Как запустить
1. Скомпилировать файл main.cpp
2. Запустить программу

## Пример функционала
Пользователь может:
- нанимать сотрудников
- увольнять
- изменять данные
- просматривать список
